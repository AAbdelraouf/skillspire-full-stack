What is the Terminal?
- Terminal allows you to interface directly with operating system (OS).
- It's faster than using the Graphic User Interface (GUI).
- The language the terminal uses is called Bash

Commands to remember
	- ls = list items 
	- cd = change directory
	- pwd = present working directory
	- .. = parent directory (one directory outside your pwd)
	- mkdir = create folder
	- touch = create file
	- / = root directory
	- ~ = home directory
	- rm = delete file
	- rm -rf = delete folder
	- mv = move/rename file or folder
