Commands to remember
	- ls = list items 
	- cd = change directory
	- pwd = present working directory
	- .. = parent directory (one directory outside your pwd)
	- mkdir = create folder
	- touch = create file
	- / = root directory
	- ~ = home directory
	- rm = delete file
	- rm -rf = delete folder
	- mv = move/rename file or folder
